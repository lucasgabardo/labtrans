# Labtrans
Desafio Labtrans.

![Labtrans](labtrans-app-gif.gif)
<h2 align="center">Install and Run</h2>

Clone the repo:
```bash
$ git clone https://github.com/lucasgabardo/labtrans.git
```

Install API dependencies and run:

labtrans-api/

```bash
$ cd labtrans-api
$ npm install 
$ npm start
```

Install Vue App dependencies and run:

labtrans-app/

```bash
$ cd labtrans-app
npm install
npm start
```
